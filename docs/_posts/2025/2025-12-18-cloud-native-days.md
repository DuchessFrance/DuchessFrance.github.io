Ce prochain 3 février 2026 se tiendra au [CENTQUATRE-PARIS](https://www.cloudnativedays.fr/lieu) la seconde édition de Kubernetes Community Days France sous le nouveau nom Cloud Native Days France (CND France).

Le rendez-vous Cloud Native et DevOps, orienté Open Source a 3 principales missions :
  - le partage des connaissances techniques
  - la veille technologique 
  - la rencontre entre les acteurs qui font cet écosystème

Leur objectif est de réunir les professionnels intéressés par le Cloud, le DevOps, Kubernetes, le Cloud Native... et leur ambition est de réunir +2k participants !

[Ici le programme en 4 tracks, annoncé sans aucun discours commercial.](https://www.cloudnativedays.fr/programme)
